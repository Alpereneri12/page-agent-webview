# 🚀 Nexus AI Browser - Mobile Agent Engine

Nexus AI, web siteleri üzerinde otonom işlemler gerçekleştirebilen, modern arayüze sahip gelişmiş bir **Mobile WebView Agent** uygulamasıdır. Kullanıcıların belirlediği web sayfalarında akıllı asistan desteği sunarak deneyimi kişiselleştirmek ve otomatize etmek için tasarlanmıştır.

## ⚡ Temel Özellikler
* **Modern Dark UI:** Glassmorphism etkileriyle donatılmış, göz yormayan profesyonel karanlık tema.
* **Smart WebView Injection:** Web sayfalarına dinamik olarak özel JS ajanları enjekte edebilme kabiliyeti.
* **Türkçe Karakter Fix:** URI encoding/decoding yapısı ile sorunsuz komut iletimi.
* **Safe Area Compatibility:** `react-native-safe-area-context` ile tüm çentikli (notch) cihazlarla tam uyum.
* **Gelişmiş Navigasyon:** WebView içi geri gitme desteği ve optimize edilmiş yükleme süreçleri.

## 🛠️ Teknik Gereksinimler & Kurulum
Proje **React Native** ve **Expo** altyapısı kullanılarak geliştirilmiştir.

1.  **Repoyu Klonlayın:**
    ```bash
    git clone [https://github.com/KULLANICI_ADIN/Nexus-AI-Browser.git](https://github.com/KULLANICI_ADIN/Nexus-AI-Browser.git)
    ```
2.  **Bağımlılıkları Yükleyin:**
    ```bash
    npm install
    npx expo install react-native-safe-area-context react-native-webview
    ```
3.  **Uygulamayı Başlatın:**
    ```bash
    npx expo start
    ```
